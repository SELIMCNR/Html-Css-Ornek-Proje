HTML Alıntı ve Alıntı Elemanları

Bu bölümde < blockquote>, < q>, < abbr>, < address>, < cite>ve < bdo>HTML öğelerini inceleyeceğiz.

Alıntılar için HTML < blockquote>


HTML < blockquote>öğesi, başka bir kaynaktan alıntılanan bir bölümü tanımlar. < blockquote>Tarayıcılar genellikle öğeleri girintiler .


Kısa Alıntılar için HTML < q>

HTML < q>etiketi kısa bir alıntıyı tanımlar. Tarayıcılar normalde alıntının etrafına tırnak işaretleri ekler.

HTML belgesinin görünen kısmı ve < body> arasındadır < /body>.


Kısaltmalar için HTML < abbr>

HTML < abbr>etiketi, "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM" gibi bir kısaltma veya kısaltma tanımlar. Kısaltmaları işaretlemek, tarayıcılara, çeviri sistemlerine ve arama motorlarına yararlı bilgiler verebilir. İpucu: Fareyi öğenin üzerine getirdiğinizde kısaltmanın/kısaltmanın açıklamasını göstermek için genel başlık özniteliğini kullanın.

İletişim Bilgileri için HTML < address> HTML < address>etiketi, bir belgenin veya makalenin yazarının/sahibinin iletişim bilgilerini tanımlar. İletişim bilgileri bir e-posta adresi, URL, fiziksel adres, telefon numarası, sosyal medya tanıtıcısı vb. olabilir.
